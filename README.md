# 🍕 Pizza Sales Data Analysis (SQL Project)

## 📌 Project Overview

This project focuses on analyzing a real-world pizza sales dataset using structured SQL queries to extract meaningful business insights.

The analysis simulates how data analysts work with relational databases to solve business problems such as identifying top-selling products, understanding customer behavior, and optimizing revenue strategies.

---

## 🎯 Objectives

* Retrieve key business metrics like total orders and total revenue
* Identify best-selling pizza types and sizes
* Analyze order patterns based on time (hourly trends)
* Understand category-wise performance
* Calculate revenue contribution and cumulative growth
* Derive actionable insights for business decision-making

---

## 🗂️ Dataset Description

The dataset consists of four relational tables:

* **orders** → Contains order date and time
* **order_details** → Contains quantity and order-wise pizza details
* **pizzas** → Contains pizza size and price
* **pizza_types** → Contains pizza names and categories
